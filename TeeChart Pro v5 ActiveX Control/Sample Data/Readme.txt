---------------------------------
  TeeChart Pro Sample Database
---------------------------------

This folder contains sample tables
necessary to run the data-aware
TeeChart Pro examples.

When installing TeeChart Pro, the setup
application automatically adds a new
database to ODBC 3.0 configuration.

The ODBC Database name is "TeeChart Pro Database",
and points to this folder. TeeChart's ADO examples
use this sample data extensively.

If you accepted the install option for 'IIS ASP 
examples' then the installer will have created a 
system DSN to facilitate db access from the ASP 
examples. The DSN is called 'TeeChart Pro System db'.

You don't need to deploy this sample database with 
your TeeChart Pro applications.
