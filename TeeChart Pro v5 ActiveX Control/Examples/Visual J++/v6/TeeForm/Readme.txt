Microsoft Visual Java++ v6 Example notes
-------------------------------------------

To create a new MS VJ++6 project using TeeChart you
need to create the Java class wrapper for the TeeChart
COM component. 
*Important:
You'll need to do step 1. from below before opening 
the example project.

Follow these steps:

1. In the Developer environment, open the menu 
option, 'Project' and 'Add COM Wrapper'.

2. Select 'TeeChart Pro Activex Control v5' from the list
of available Activex controls.

The VJ++6 Developer environment will create a folder of all 
TeeChart classes below (as default) the project folder.

3. You need to compile the project once before VJ++ will
recognise the TeeChart classes or before you are able to 
add a TeeChart component to a Form.

5. After compiling for the first time you may select a TeeChart
component from the Toolbox and place it on a Form. VJ++6 will
create the necessary container code for each component.
---------------------------------------------

For reference purposes you may use the VJ++6 Object browser to 
consult property and method declarations. Alternatively, use
TeeChart's online helpfile.
